# テクスチャ作成指示書テンプレート（texture_request_template）

新しいテクスチャをClaude / Claude Codeに依頼する際は、本テンプレートをコピーして必要事項を埋めること。
共通規格は `map_texture_standard.md` に従う（本指示書で規格を上書きしない）。

---

## 依頼内容

- **テクスチャ名**: （命名規則に従う。例: `terrain_sand_top_01`）
- **分類**: terrain / prop / char / fx
- **サイズ**: （規格書のデフォルトなら「規格通り」と記載）
- **用途**: （どのアセット・どの面に貼るか。例: 砂漠マップの上面キューブ）

## 見た目の指定

- **ベースパレット**:
  - palette.json の既存エントリを使う場合: エントリ名（例: `grass`）
  - 新規の場合: 5〜6色のHEXを列挙（例: `#d8c07a, #c9b06a, #e2cd8d, #baa055, #cfb877`）
- **特徴要素**: （該当するものを記述。なければ「なし」）
  - 例: ひび割れ / 発光コア / 上端の草垂れ / 小石の散らばり / 波紋
- **ノイズの粒度**: 細かい / 標準 / 粗い（迷ったら「標準」。ゲーム内カメラで確認して調整）

## 制約

- **シームレス**: 必須 / 不要
- **透過**: なし / あり（対象箇所を明記）
- **アニメーション前提**: なし / あり（あり→シェーダーでUVスクロール等。テクスチャ側の対応を記載）

## バリエーション

- **必要枚数**: （例: `_01`〜`_03` の3枚。シード違いで生成）

## 参考

- （参考画像・既存テクスチャ・ゲームタイトル等があれば記載）

## 完了条件

- [ ] `tools/asset_gen/` のスクリプトから再生成可能な状態で実装されている
- [ ] palette.json に色定義が追加されている（新規パレットの場合）
- [ ] プレビューシート（preview_sheet.png）で目視確認済み
- [ ] （terrainの場合）対応するGLBアセットの再生成まで完了している

## 記入例

- テクスチャ名: `terrain_sand_top_01`
- 分類: terrain / サイズ: 規格通り / 用途: 砂漠地形の上面
- ベースパレット: 新規 `sand` → `#d8c07a, #c9b06a, #e2cd8d, #baa055, #cfb877`
- 特徴要素: 小石の散らばり（stone系の暗色を数ピクセル）
- ノイズの粒度: 粗い
- シームレス: 必須 / 透過: なし / アニメーション前提: なし
- バリエーション: `_01` のみ
