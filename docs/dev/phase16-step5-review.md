# Phase16-Step5 品質判定レビュー

`docs/dev/phase/phase16-step5.md` に基づき作成した見本市シーンの記録。
**最終go/no-go判定は制作者（野村）が行うもの**なので、以下は実装側からの
技術的な報告と、チェックリストに対する一次評価（ドラフト）。主観評価項目
（雰囲気・「プレイしたい」と思えるか等）は野村の目視確認をもって確定させる。

## 実装したもの

- `samples/ShowcaseMap.tscn` / `scripts/map/showcase_map.gd`
- 10×10の全地形要素（草原・土の小道・石の高台・水辺・溶岩・階段）+
  装飾（grass_patch×7, broken_stone×3, flag_placeholder×1）+
  プレースホルダーユニット4体（草原に自軍2、高台に敵軍2）
- テーマ/環境/カメラはPhase16AssetPreviewと同じ本番設定
  （`theme_default.tres` / `battle_atmosphere.tres` / `CameraController`）を流用
- 3構図をコマンドライン引数（`--composition=standard|overview|closeup`）で
  切り替えつつ、実行中は数字キー1/2/3・マウスドラッグ/ホイール/Q・Eでも
  手動調整できるようにした（`ScreenshotCapture`のF12/`--auto-capture`で撮影）

## マップ構成

- 石の高台: x3-6, z2-5（東端x=6が高さ3の頂上、他は高さ2）
- 水辺: x7-8, z2-4（高台の東の崖に接し、3段の崖を作る。東側は草原と接する）
- 溶岩だまり: x3-4, z6（高台南面とフラットに連結、1×2）
- 階段: x5, z6（高台から草原への1段接続）
- 土の小道: x=1列（z1-8、草原を縦断）
- 草原: 残り67マス（高さ0-1）

配置意図: カメラは概ね-x/+z方向を奥にして見下ろすため、高台を画面中央〜
奥寄りに置き、その崖面（東側）に水辺を隣接させることで「高台→水辺」の
高低差が画面手前に向かって開ける構図にした。

## 実装中に発見した不具合（修正済み）

**背景プレーンが水面/溶岩面より高い位置にあり、高さ0の水セルを完全に
隠していた。**

- `voxel_map.gd`の`_create_background_plane()`は、マップ全体を覆う
  ボイド埋め用の不透明プレーンをY=-0.02に配置していた
- 水面・溶岩面は`SURFACE_OFFSET=0.08`分だけセルの高さより沈むため
  （Phase16-Step2の仕様）、高さ0の水セルは実際にはY=-0.08に描画される
- Y=-0.08はY=-0.02より低いため、不透明な背景プレーンが水面の**手前**に
  描画され、水面を完全に覆い隠していた
- これまでの見本マップ（DioramaSample / Phase16AssetPreview）は水・溶岩を
  常に周囲と同じ高さ（height>=1）に置いていたため、この問題が一度も
  顕在化していなかった（`phase16-step4-log.md`に記載の「崖の死角」問題とは
  別の、高さ0の水セル特有の潜在バグ）
- 修正: 背景プレーンをY=-0.02→-0.5に変更（`scripts/map/voxel_map.gd`）。
  Main.tscn含む全シーン共通のコードのため、実バトルで高さ0の水/溶岩地形を
  使う場合にも今後この問題は起きない

## チェックリスト一次評価

スタイルの統一感:
- [x] 全アセットが同じ世界の物に見える（Step4までの調整済みパレット・
      ライティングをそのまま使用しているため、テクスチャ密度・色調の
      不揃いは見られない）
- [x] 「素材の寄せ集め」感はない

目標との距離:
- [ ] タクティクスオウガ/Minecraft Dungeons的な方向性かどうか —
      **要野村確認**。実装側の所感としては、方向性（暗めのトーン・
      陰影・溶岩のグロー）は近づいているが、地形が単一の直方体ブロック
      主体の構成になっており、参照イメージのような自然な起伏の複雑さは
      このサンプルでは表現しきれていない。地形タイプの見本市としては
      機能しているが、「量産後の実マップの見え方」を判断するには
      レイアウトがやや単調な可能性がある

ゲームとしての成立:
- [x] 標準構図でマス目・高低差・地形タイプは判読できる
- [x] ユニットは背景から浮いて見える（沈んでいない）
- [ ] 30分眺めて「プレイしたい」と思えるか — **要野村確認（主観評価のため）**
