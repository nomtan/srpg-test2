# ステータス・Build値仕様

## 1. 基本方針

本作では、キャラクターが持つ成長・装備・スキル補正の対象となる値を「基礎ステータス」と呼ぶ。

一方で、戦闘中に実際のダメージ計算・命中判定・行動順・移動処理などで参照される最終評価値を「Build値」と定義する。

---

## 2. 基礎ステータス

キャラクターは以下の基礎ステータスを持つ。

| ステータス | 内容 |
|---|---|
| STR | 筋力。物理攻撃力に影響する |
| DEX | 器用さ。命中率・会心率に影響する |
| VIT | 耐久力。防御力・HPに影響する |
| MND | 精神力。魔法防御力・回復力・状態異常耐性に影響する |
| INT | 知力。魔法攻撃力に影響する |
| AGI | 敏捷性。素早さ・移動量・跳躍量・回避率に影響する |

---

## 3. Build値

Build値とは、戦闘中に実際の処理で参照される最終評価値である。

| Build値 | 算出元 | 内容 |
|---|---|---|
| 攻撃力 | STR | 物理攻撃・物理スキルの威力に使用する |
| 魔法攻撃力 | INT | 魔法攻撃・属性スキルの威力に使用する |
| 防御力 | VIT | 物理ダメージの軽減に使用する |
| 魔法防御力 | MND | 魔法ダメージの軽減に使用する |
| 素早さ | AGI | CT蓄積速度・行動順に使用する |
| 移動量 | AGI | 1回の移動で進めるマス数に使用する |
| 跳躍量 | AGI | 登れる段差・越えられる高低差に使用する |
| 命中率 | DEX | 攻撃・スキルの命中判定に使用する |
| 会心率 | DEX | クリティカル発生判定に使用する |

---

## 4. ステータス補正ルール

基礎ステータスは以下の要素によって増減する。

- キャラクター成長
- ジョブ補正
- 装備補正
- パッシブスキル補正
- バフ効果
- デバフ効果

Build値は、最終的な基礎ステータスから算出する。

---

## 5. 計算順

ステータスおよびBuild値は以下の順番で計算する。

1. キャラクター基礎値を取得
2. ジョブ補正を加算
3. 装備補正を加算
4. パッシブスキル補正を加算
5. バフ・デバフ補正を加算
6. 最終ステータスを算出
7. 最終ステータスからBuild値を算出

---

## 6. 最終ステータスの考え方

各基礎ステータスは、以下のように算出する。

```txt
最終ステータス = キャラクター基礎値 + ジョブ補正 + 装備補正 + パッシブ補正 + バフ・デバフ補正
```

## 7. Build値の計算式

### 攻撃力

攻撃力はSTRから算出する。

```txt
攻撃力 = 最終STR
```

攻撃力は以下に使用する。

- 通常物理攻撃
- 物理スキル
- STR依存の武器攻撃
- 一部の地形破壊スキル

---

### 魔法攻撃力

魔法攻撃力はINTから算出する。

```txt
魔法攻撃力 = 最終INT
```

魔法攻撃力は以下に使用する。

- 攻撃魔法
- 属性魔法
- INT依存のスキル
- 魔法系の地形変化スキル

---

### 防御力

防御力はVITから算出する。

```txt
防御力 = 最終VIT
```

防御力は以下に使用する。

- 物理ダメージ軽減
- 近接攻撃への耐久
- 一部のノックバック耐性
- 一部の地形ダメージ軽減

---

### 魔法防御力

魔法防御力はMNDから算出する。

```txt
魔法防御力 = 最終MND
```

魔法防御力は以下に使用する。

- 魔法ダメージ軽減
- 属性ダメージ軽減
- 状態異常耐性
- 回復魔法の効果量補正

---

### 素早さ

素早さはAGIの影響を強く受ける。

```txt
素早さ = 最終AGI
```

ジョブ補正を入れる場合は以下とする。

```txt
素早さ = floor(最終AGI × ジョブ素早さ補正)
```

素早さは以下に使用する。

- CT蓄積速度
- 行動順
- 待機後の次回行動順
- 一部の回避判定

---

### 移動量

移動量はAGIが一定値に達するごとに上昇する。

```txt
移動量 = 基本移動量 + floor(最終AGI / 20)
```

#### AGIによる移動量補正例

| 最終AGI | 移動量補正 |
|---:|---:|
| 0〜19 | +0 |
| 20〜39 | +1 |
| 40〜59 | +2 |
| 60〜79 | +3 |
| 80〜99 | +4 |

---

### 跳躍量

跳躍量は移動量よりも伸びにくくする。

```txt
跳躍量 = 基本跳躍量 + floor(最終AGI / 30)
```

#### AGIによる跳躍量補正例

| 最終AGI | 跳躍量補正 |
|---:|---:|
| 0〜29 | +0 |
| 30〜59 | +1 |
| 60〜89 | +2 |
| 90〜119 | +3 |

---

### 命中率

命中率はDEXの影響を比較的強く受ける。

```txt
命中率 = 基本命中率 + floor(最終DEX / 2)
```

#### DEXによる命中率補正例

| 最終DEX | 命中率補正 |
|---:|---:|
| 10 | +5% |
| 20 | +10% |
| 30 | +15% |
| 40 | +20% |
| 50 | +25% |

命中率は以下に使用する。

- 通常攻撃の命中判定
- 物理スキルの命中判定
- 弓・短剣・銃などの精密武器の命中判定
- 一部の状態異常付与スキルの成功率

---

### 会心率

会心率はDEXの影響を受けるが、伸びすぎないようにする。

```txt
会心率 = 基本会心率 + floor(最終DEX / 5)
```

#### DEXによる会心率補正例

| 最終DEX | 会心率補正 |
|---:|---:|
| 10 | +2% |
| 20 | +4% |
| 30 | +6% |
| 40 | +8% |
| 50 | +10% |

会心率は以下に使用する。

- 通常攻撃のクリティカル判定
- 物理スキルのクリティカル判定
- 弓・短剣・双剣などの軽量武器の会心判定
- 一部のパッシブスキル発動条件

---

## 8. HP算出式

HPはVITを主軸にし、ジョブごとの耐久差を出す。

```txt
最大HP = 基本HP + floor(最終VIT × 5) + ジョブHP補正
```

### 例

```txt
基本HP = 100
最終VIT = 30
ジョブHP補正 = 20

最大HP = 100 + floor(30 × 5) + 20
最大HP = 270
```

### 方針

- VITが高いほどHPが高くなる
- 重騎士・格闘師などはジョブHP補正を高くする
- 魔術師・治癒師などはジョブHP補正を低くする

---

## 9. 回避率

回避率はAGIを主軸にし、DEXを少しだけ補助として使う。

```txt
回避率 = 基本回避率 + floor(最終AGI / 3) + floor(最終DEX / 10)
```

### 例

```txt
基本回避率 = 5%
最終AGI = 30
最終DEX = 20

回避率 = 5 + floor(30 / 3) + floor(20 / 10)
回避率 = 17%
```

### 方針

- AGI型は回避に強い
- DEXは回避補助として少しだけ影響する
- 回避率が高くなりすぎないように、上限値を設定する

```txt
回避率上限 = 60%
```

---

## 10. 状態異常耐性

状態異常耐性はMNDを主軸にする。

```txt
状態異常耐性 = 基本耐性 + floor(最終MND / 2)
```

### 例

```txt
基本耐性 = 0%
最終MND = 30

状態異常耐性 = 0 + floor(30 / 2)
状態異常耐性 = 15%
```

---

### 状態異常成功率

状態異常付与スキルは、使用者のDEXまたはINTと、対象のMNDを比較して成功率を決める。

物理系・技巧系状態異常の場合：

```txt
状態異常成功率 = スキル基本成功率 + floor(使用者のDEX / 3) - floor(対象のMND / 2)
```

魔法系状態異常の場合：

```txt
魔法状態異常成功率 = スキル基本成功率 + floor(使用者のINT / 3) - floor(対象のMND / 2)
```

### 方針

- MNDが高いキャラは状態異常に強い
- DEX型は毒・出血・罠などの状態異常付与が得意
- INT型は呪い・沈黙・睡眠などの魔法状態異常が得意

---

## 11. ダメージ計算式

物理ダメージは、攻撃力と防御力を使って計算する。

```txt
物理ダメージ = max(1, 攻撃力 + スキル威力 - 防御力)
```

### 例

```txt
攻撃力 = 35
スキル威力 = 20
防御力 = 25

物理ダメージ = max(1, 35 + 20 - 25)
物理ダメージ = 30
```

### 会心発生時

```txt
会心ダメージ = 物理ダメージ × 会心倍率
```

```txt
会心倍率 = 1.5
```

### 方針

- STRが高いほど物理ダメージが伸びる
- VITが高いほど物理ダメージを軽減する
- スキル威力で技ごとの差を出す
- 最低保証ダメージは1とする

---

## 12. 魔法ダメージ計算式

魔法ダメージは、魔法攻撃力と魔法防御力を使って計算する。

```txt
魔法ダメージ = max(1, 魔法攻撃力 + スキル威力 - 魔法防御力)
```

### 例

```txt
魔法攻撃力 = 40
スキル威力 = 25
魔法防御力 = 30

魔法ダメージ = max(1, 40 + 25 - 30)
魔法ダメージ = 35
```

### 属性補正込み

```txt
最終魔法ダメージ = 魔法ダメージ × 属性相性補正 × 地形補正
```

### 方針

- INTが高いほど魔法ダメージが伸びる
- MNDが高いほど魔法ダメージを軽減する
- 属性相性や地形補正でダメージ差を出す

---

## 13. 回復量計算式

回復量はMNDを主軸にする。

```txt
回復量 = スキル回復力 + floor(最終MND × 1.5)
```

### 例

```txt
スキル回復力 = 30
最終MND = 24

回復量 = 30 + floor(24 × 1.5)
回復量 = 66
```

### 光・水・風などの属性回復

水属性・光属性、風属性などの回復スキルは、地形や状態によって補正を受ける。

```txt
最終回復量 = 回復量 × 地形補正 × 状態補正
```

### 方針

- MND型は回復役として強い
- 光・水属性キャラは回復スキルと相性が良い
- 呪い地形・闇霧などでは回復量が下がる
- 聖域・水場などでは回復量が上がる

| 杖 | INT | MND | 魔法攻撃・補助 |
| 鈍器 | STR | MND | 物理＋聖職者向き |
| 格闘 | STR | AGI | 近接・連撃向き |


```txt

```

#### 弓

```txt

```

#### 短剣

```txt

```

```txt

```

## 15. スキル分類

スキルは大きく以下に分類する。

| 分類 | 内容 |
|---|---|
| 物理技 | STR・DEX・AGIなどを参照する攻撃スキル |
| 魔法 | INT・MNDを参照する攻撃・補助スキル |
| 回復 | MNDを参照する回復スキル |
| 地形スキル | タイルの地形・高さ・オブジェクト・罠を変化させるスキル |
| パッシブスキル | 常時発動する補正スキル |
| リアクションスキル | 攻撃を受けた時などに発動する反応スキル |
| サポートスキル | 味方強化・敵弱体・移動補助など |
| コンボスキル | 地形や属性条件を満たすと発動する連携スキル |

---

### 物理技

```txt
参照値 = 攻撃力 または 武器攻撃力
```

例：

- 強斬り
- 兜割り
- 二連斬
- 貫通突き
- 狙撃

---

### 魔法

```txt
参照値 = 魔法攻撃力
```

例：

- 火球
- 雷撃
- 闇霧
- 光柱
- 氷結

---

### 回復

```txt
参照値 = MND / 回復量
```

例：

- 治癒
- 範囲回復
- 状態異常解除
- 聖域回復

---

### 地形スキル

```txt
参照値 = スキル固有値
```

地形スキルはダメージよりも、タイル変化・戦場制御を主目的とする。

例：

- 隆起
- 岩盤化
- 水場生成
- 炎上タイル
- 闇霧
- 聖域化

---

| 炎上 | 風 | 火炎旋風として範囲拡大 |
| 炎上 | 水 | 消火・蒸気爆発 |
| 荒地 | 水 | 泥沼化 |
| 岩盤 | 火 | 溶岩化 |
| 闇霧 | 光 | 浄化閃光 |
| 呪い地形 | 光 | 聖域化 |
| 聖域 | 闇 | 侵食崩壊 |

---

### 地形ダメージ

```txt
地形ダメージ = 地形基本ダメージ - floor(対応防御値 / 4)
```

例：

```txt
炎上ダメージ = 火地形基本ダメージ - floor(MND / 4)
溶岩ダメージ = 溶岩基本ダメージ - floor(VIT / 4)
呪いダメージ = 呪い基本ダメージ - floor(MND / 4)
```

---

## 19. 命中判定

命中判定は、攻撃側の命中率と、防御側の回避率を比較して決定する。

```txt
最終命中率 = 攻撃側命中率 - 防御側回避率 + スキル命中補正 + 地形補正 + 高低差補正
```

### 下限・上限

```txt
最終命中率下限 = 5%
最終命中率上限 = 95%
```

### 例

```txt
攻撃側命中率 = 85%
防御側回避率 = 20%
スキル命中補正 = 0%
地形補正 = 0%
高低差補正 = +10%

最終命中率 = 85 - 20 + 0 + 0 + 10
最終命中率 = 75%
```

---

## 20. 会心判定

会心判定は、攻撃が命中した後に行う。

```txt
最終会心率 = 会心率 + 武器会心補正 + スキル会心補正 - 対象会心耐性
```

### 下限・上限

```txt
最終会心率下限 = 0%
最終会心率上限 = 50%
```

### 会心ダメージ

```txt
会心ダメージ = 通常ダメージ × 1.5
```

### 方針

- DEX型は会心率が高い
- 短剣・弓・双剣は会心補正が高い
- 重武器は会心率よりも基礎威力を重視する

## 21. CT・素早さ連動

素早さはCT蓄積速度に影響する。

```txt
CT増加量 = 素早さ
```

CTが100以上になったユニットから行動可能になる。

```txt
CT >= 100 で行動可能
```

### 待機時

待機を選択した場合、次回CTにボーナスを与える。

```txt
待機CTボーナス = +20
```

### 方針

- AGIが高いキャラは行動順が早い
- 待機によって次回行動を早められる
- 鈍足・加速などのバフデバフが行動順に大きく影響する

## 23. バランス上の注意点

### AGIの万能化を防ぐ

AGIは以下の複数要素に影響する。

- 素早さ
- 移動量
- 跳躍量
- 回避率

そのため、移動量と跳躍量はAGIに比例して直接伸ばさず、一定値ごとに段階的に上昇させる。

---

### DEXの価値を明確にする

DEXは攻撃力そのものには影響しない。

ただし、以下に影響することで、技術系・精密系ジョブの価値を作る。

- 命中率
- 会心率
- 一部武器の安定性
- 一部状態異常スキルの成功率

---

### STRとDEXの役割を分ける

STRは「当たったときの威力」に影響する。

DEXは「当てる力」と「急所を突く力」に影響する。

そのため、以下のような差別化が可能になる。

| 型 | 特徴 |
|---|---|
| STR型 | 高威力だが命中・会心は控えめ |
| DEX型 | 威力は控えめだが命中・会心が高い |
| AGI型 | 行動順と移動性能に優れる |
| VIT型 | 前線維持に優れる |
| INT型 | 魔法火力に優れる |
| MND型 | 回復・魔法耐久・状態異常耐性に優れる |
